# Matthew Park - Portfolio Website

A clean, minimal portfolio website built with React and Tailwind CSS.

## Technologies

- React 18
- Vite
- Tailwind CSS
- Responsive Design

## Getting Started

Install dependencies:
```bash
npm install
```

Run development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

Preview production build:
```bash
npm run preview
```

## Deployment

This project is ready to be deployed to Vercel. Simply push to GitHub and connect your repository to Vercel.
